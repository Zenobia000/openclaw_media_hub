參加工作坊前，您需要準備與安裝以下項目：

**一、 必備硬體與軟體安裝**

- **筆記型電腦**：需為 macOS / Windows 10+ / Linux 作業系統，且**至少具備 8GB RAM**。
- **Docker Desktop**：必須預先安裝好。
- **Telegram App**：手機端需安裝好 App，並完成帳號註冊。

**二、 必備註冊帳號與網路**

- 穩定的網路連線。
- **GitHub 帳號**（免費）。
- **Anthropic 帳號**
- **Google 帳號**（後續需啟用 Gmail + Calendar API）。
- **Resend 帳號**（免費，用於發信）。

**三、 建議準備（非必要項目）**

- **VS Code 編輯器**：建議安裝，主要用於查看設定檔。
- **Notion 帳號**：如果您預計選擇 Notion 作為 CRM 客戶管理系統的資料儲存位置，建議預先註冊。

課前一天，講師也會發送環境檢查腳本（setup-check.sh / setup-check.ps1）協助您確認所有工具是否正確安裝，並提供線上 Office Hour 協助排除安裝問題。
