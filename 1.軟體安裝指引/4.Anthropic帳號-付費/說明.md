# Anthropic 簡介

Anthropic 是一家由前 OpenAI 高層創立的 AI 安全與研究公司，致力於開發可靠、可解釋且安全的 AI 系統。其旗艦模型 **Claude** 系列在目前 AI 市場中表現極為出色。

## 為什麼選擇 Anthropic (Claude)？

1. **卓越的程式碼能力**：Claude 3.5 Sonnet 模型在撰寫、除錯與重構程式碼方面的表現被公認為目前界內頂尖，非常適合開發者與學員使用。
2. **強大的邏輯推理**：相較於其他模型，Claude 在處理複雜邏輯、遵循指令及語意理解上更為細膩與準確。
3. **Artifacts 功能**：提供即時的程式碼預覽與文件編輯環境，能大幅提升教學與學習的互動性。

## 付費建議

有免費的替代方案（如 Claude 的免費版或其他平台），但是免費資源往往有嚴格的使用限制（如每小時對話次數），且 AI 回應的穩定性參差不齊，可能會影響老師教學進度與學員學習體驗。

**建議學員可以考慮自費訂閱 Claude Pro**，以獲取最強大的模型效能與更穩定的使用頻寬。

## 註冊與訂閱流程

### 建立帳號

1. 前往 [Anthropic 官方網站](https://www.anthropic.com/)。

2. 點擊使用 Google 帳號繼續登入。

   ![alt text](image.png)

3. 確認帳號資訊後，點擊「繼續」。

   ![alt text](image-2.png)

4. 填寫使用者名稱，勾選同意隱私政策，點擊「建立帳號」。

   ![alt text](image-3.png)

5. 進行手機號碼驗證。台灣手機號碼需將開頭的 `09XX` 改為國際格式 `+8869XX`，填寫後點擊「驗證」。

   ![alt text](image-4.png)

6. 輸入收到的簡訊驗證碼，點擊「驗證」完成認證。

   ![alt text](image-5.png)

7. 建議關閉「使用聊天紀錄訓練模型」選項，然後點擊「I understand」。

   ![alt text](image-6.png)

8. 職業欄位可依個人狀況填寫，不影響後續使用，填寫後點擊「繼續」。

   ![alt text](image-7.png)

9. 點擊「繼續」進入下一步。

   ![alt text](image-8.png)

### 訂閱 Pro 方案

10. 選擇購買一個月的 Pro 方案，點擊「Get Pro plan」。
    - 注意：費用為每月 20 美金（約新台幣 650 元）
    - 注意：主要用於使用 Claude Code 功能

    ![alt text](image-9.png)

11. 填寫信用卡付款資訊，完成訂閱。

    ![alt text](image-10.png)

12. 前往 [Claude 聊天介面](https://claude.ai/new)，於左下角確認顯示「Pro plan」即表示訂閱成功。

    ![alt text](image-11.png)

### 安裝 Claude Code

13. 開啟終端機，依照您使用的環境選擇對應的安裝指令。
    [官方安裝說明](https://code.claude.com/docs/en/overview)

    1. **PowerShell**：貼上以下指令後按 Enter。

       ```bash
       irm https://claude.ai/install.ps1 | iex
       ```

       ![alt text](image-12.png)

    2. **命令提示字元**：貼上以下指令後按 Enter。

       ```bash
       curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
       ```

       ![alt text](image-13.png)

14. 在桌面建立一個新資料夾，例如命名為 `claude`。

    ![alt text](image-14.png)

15. 複製該資料夾的路徑（在檔案總管的路徑列空白處點擊一下，再按 Ctrl+C 複製）。

    ![alt text](image-15.png)

16. 在終端機中使用 `cd` 指令切換至該資料夾。將複製的路徑貼上後按 Enter。
    - 注意：`cd` 與路徑之間需保留一個空格
    - `cd` 指令用途：切換至指定的資料夾

    ```
    cd 路徑
    ```

    當終端機顯示的路徑與您貼上的路徑一致時，即表示切換成功。

    ![alt text](image-16.png)

17. 在終端機中輸入 `claude code`，出現授權同意畫面後按 Enter 確認。

    ![alt text](image-17.png)

18. 出現以下畫面即表示安裝成功，確認後可關閉視窗。
    - **Sonnet 4.6**：目前選用的 AI 模型
    - **Claude Pro**：目前使用的訂閱方案

    ![alt text](image-18.png)
