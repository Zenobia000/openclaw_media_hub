# Cron 表達式速查表

## 格式

```
┌───────────── 分鐘 (0-59)
│ ┌───────────── 小時 (0-23)
│ │ ┌───────────── 日期 (1-31)
│ │ │ ┌───────────── 月份 (1-12)
│ │ │ │ ┌───────────── 星期 (0-7，0 和 7 都是星期日)
│ │ │ │ │
* * * * *
```

## 特殊字元

| 字元 | 意義 | 範例 |
|------|------|------|
| `*` | 任意值 | `* * * * *`（每分鐘） |
| `,` | 列舉 | `0,30 * * * *`（每小時的 0 分和 30 分） |
| `-` | 範圍 | `0 9-17 * * *`（9:00 到 17:00 的每小時整點） |
| `/` | 間隔 | `*/15 * * * *`（每 15 分鐘） |

## 常用範例

| 說明 | cron 表達式 | 備註 |
|------|------------|------|
| 每天早上 8 點 | `0 8 * * *` | 日報排程 |
| 每天早上 9 點 | `0 9 * * *` | 工作開始提醒 |
| 週一到週五早上 9 點 | `0 9 * * 1-5` | 工作日執行 |
| 每週五下午 5 點 | `0 17 * * 5` | 週報排程 |
| 每小時整點 | `0 * * * *` | 行程提醒 |
| 每 30 分鐘 | `*/30 * * * *` | 頻繁檢查 |
| 每月 1 號早上 10 點 | `0 10 1 * *` | 月報排程 |
| 每天早上 8 點和下午 6 點 | `0 8,18 * * *` | 早晚各一次 |
| 週一早上 9 點 | `0 9 * * 1` | 每週開始提醒 |
| 每天凌晨 2 點 | `0 2 * * *` | 批次作業 |

## 星期對照

| 數字 | 星期 |
|------|------|
| 0 | 星期日 |
| 1 | 星期一 |
| 2 | 星期二 |
| 3 | 星期三 |
| 4 | 星期四 |
| 5 | 星期五 |
| 6 | 星期六 |
| 7 | 星期日（同 0） |

## 組合範例

```
# 工作日每小時（9-17 點）執行
0 9-17 * * 1-5

# 每天 8 點、12 點、18 點
0 8,12,18 * * *

# 每月 1 號和 15 號早上 10 點
0 10 1,15 * *

# 每季第一個月的 1 號
0 10 1 1,4,7,10 *
```
