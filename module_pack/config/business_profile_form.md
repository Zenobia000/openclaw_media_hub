## 填完之後

填完這份表單後，使用 Claude Code 執行以下命令，系統會自動根據你的設定修改所有模板：

```bash
# 安裝預約模組
/install-booking

# 安裝 Email 模組
/install-email

# 安裝 CRM 模組
/install-crm

# 安裝客服模組
/install-customer-service

# 安裝報告模組
/install-report

# 安裝通知模組
/install-notify
```

每個 `/install-xxx` 命令會：
1. 讀取你在本表單填寫的設定
2. 自動修改對應模組的 CLAUDE.md、skill config、prompt 模板
3. 產出一份安裝確認報告，列出所有被修改的檔案與設定值

**你不需要手動修改任何技術檔案。** 所有技術設定都由 Claude Code 根據這份表單自動完成。
