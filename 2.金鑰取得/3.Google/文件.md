# Google Calendar

## 第一步：啟用服務

1. 進入[Google Cloud Console](https://console.cloud.google.com/)

2. 點擊左上角三條線選單

![alt text](image.png)

3. 選擇「API 和服務」>「程式庫」。

![alt text](image-1.png)

### Calendar API

4. 搜尋「Calendar API」，點擊進入。

![alt text](image-2.png)

5. 點擊「Google Calendar API」選項。

![alt text](image-3.png)

6. 點擊啟用。

![alt text](image-4.png)

### Gmail API

4. 搜尋「Gmail API」，按enter。

![alt text](image-16.png)

5. 點擊「Gmail API」選項。

![alt text](image-17.png)

6. 點擊啟用。

### Google Drive API

4. 搜尋「Google Drive API」，按enter。
   ![alt text](image-20.png)

5. 點擊「Google Drive API」選項。

![alt text](image-19.png)

6. 點擊啟用。

![alt text](image-21.png)

## 第二步：設定 OAuth 同意畫面

1. 左側點擊「OAuth 同意畫面」。

![alt text](image-5.png)

2. 點擊開始

![alt text](image-6.png)

3. 填寫應用程式名稱(隨意填)與郵件

![alt text](image-7.png)

4. 選擇「外部」

![alt text](image-8.png)

5. 填寫email

![alt text](image-9.png)

6. 勾選同意後建立

![alt text](image-10.png)

7. 目標對象 > 新增測試使用者 > 輸入自己的信箱 > 儲存

![alt text](image-15.png)

## 第三步：建立憑證

1. 點擊左上角三條線選單

![alt text](image.png)

2. 選擇「API 和服務」>「憑證」。

![alt text](image-11.png)

3. 點擊「建立憑證」>「OAuth 用戶端 ID」。

![alt text](image-12.png)

4. 選擇「電腦版應用程式」，點擊建立。

![alt text](image-13.png)

5. 下載 JSON 檔案

![alt text](image-14.png)
