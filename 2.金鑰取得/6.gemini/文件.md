# Gemini API 金鑰取得

## 金鑰用途

Gemini API Key 是存取 Google Gemini AI 模型的身份驗證憑證。Gemini 是 Google 推出的多模態 AI 模型，具備文字理解、程式碼生成、圖像分析等能力。透過此金鑰，應用程式可以呼叫 Gemini API 實現智慧對話、內容生成與資料分析等功能。

---

## 取得流程

1. 進入 [Google AI Studio API Keys 頁面](https://aistudio.google.com/api-keys)，點擊「Create API key」。

   ![alt text](image.png)

2. 填寫 API 名稱，選擇要綁定的 Google Cloud 專案。若無現有專案，點擊「建立專案」。

   ![alt text](image-1.png)

3. 填寫新專案名稱後，點擊「建立專案」完成建立。

   ![alt text](image-2.png)

4. 確認專案選擇無誤後，點擊「建立 Key」。

   ![alt text](image-3.png)

5. 金鑰建立完成後，複製顯示的 API Key 並妥善保存。

   ![alt text](image-4.png)
