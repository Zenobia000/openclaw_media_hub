# Resend API 金鑰取得

## 金鑰用途

Resend API Key 是呼叫 Resend 郵件服務的身份驗證憑證。透過此金鑰，應用程式可以使用 Resend API 發送電子郵件、追蹤郵件狀態，實現自動化的郵件通知功能。

---

## 取得流程

1. 前往 [Resend API Keys 管理頁面](https://resend.com/api-keys)，點擊「Create API Key」建立新金鑰。

   ![alt text](image.png)

2. 填寫金鑰名稱（可自行命名），權限（Permissions）選擇「Full Access」，點擊「Add」確認建立。

   ![alt text](image-1.png)

3. 金鑰建立完成後，複製顯示的 API Key 並妥善保存。請注意，此金鑰僅會顯示一次，離開頁面後將無法再次查看。

   ![alt text](image-2.png)
