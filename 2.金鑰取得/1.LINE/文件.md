# 建立LINE Bot

1. 進入[LINE官方帳號](https://manager.line.biz/)

2. 點擊聊天

![alt text](image-14.png)

3. 點擊Message API

![alt text](image-15.png)

4. 啟動Message API

![alt text](image-16.png)

5. 選擇剛剛建立的官方帳號名稱，並點擊同意

![alt text](image-17.png)

6. 點擊確定

![alt text](image-18.png)

7. 點擊確定

![alt text](image-19.png)

8. 到[開發者人員頁面](https://developers.line.biz/en/)，點擊右上角console

![alt text](image.png)

10. 在 providers 中點擊自己的專案

![alt text](image-1.png)

11. 點擊 Messaging API

![alt text](image-2.png)

12. 在Basic settings頁面中往下滑，找到 Channel secret，並點擊複製，開筆記本，貼上。

![alt text](image-3.png)

![alt text](image-4.png)

13. 在Messaging API頁面中往下滑，找到 Channel access token，點擊Issue按鈕，複製產生的token，開筆記本，貼上。

![alt text](image-5.png)

![alt text](image-6.png)

![alt text](image-7.png)

14. 紀錄此頁面的網址，之後會需要在此頁面設定Webhook URL。

- 黃色區塊之後才需要設定
  ![alt text](image-8.png)
