### 品質指標

- **測試覆蓋率**: 主力業務邏輯 (Python subprocess) 需達 80%。
- **穩定性**: PyInstaller 打包出的檔案，在 Windows/Linux 原生環境啟動不可直接閃退，崩潰率 (Crash Rate) < 1%。
- **流暢度**: 即時 Log 顯示延遲 < 500ms，不會造成整個 UI 凍結。
